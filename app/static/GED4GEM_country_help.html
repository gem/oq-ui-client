GED4GEM Country Facts Documentation

This tool is intended to allow users to select aggregate exposure information

The application is set up to be used in two ways:

    1. Under Layers, one can select one or more country year, then use the get feature info tool: and click on a country on in the map, you will get it's respective information.
    2. By opening the second accordion tab (bottom left region of the tool) labeled 'Search by Country Name' and opening up the southern grid by clicking on the collapse button: . Next enter a country name and a year, the southern grid will then be populated with the requested information.
General mapping tips and tricks:
    One can also zoom in to a region by holding down the shift key and then clicking and dragging a box on the map.
    In the southern grid, select a row, then use the zoom tool: in the top tool bar to zoom to the region.
